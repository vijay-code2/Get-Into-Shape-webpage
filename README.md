Get Into Shape

Get Into Shape is a beginner-focused fitness webpage designed to address common questions and challenges individuals face at the start of their fitness journey. The platform provides clear, accessible information to help users make informed decisions about training and health.

Features

Workout guidance for beginners

Training splits and exercise structure

Information on weight gain and weight loss

User-friendly and intuitive layout

Technologies Used

HTML – Structure and content

CSS – Styling and responsive design

JavaScript – Interactivity and dynamic behavior

Purpose

The goal of this project is to offer a simple, informative, and easy-to-navigate resource for users who are new to fitness, while also demonstrating front-end web development skills.
